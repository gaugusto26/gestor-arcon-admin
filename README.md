## 📋 Sobre o Projeto

A **NEWSOFTWARE** é uma plataforma SaaS desenvolvida para automatizar todo o ciclo de gestão de empresas de software:

- Gestão de clientes
- Contratos digitais
- Faturamento recorrente
- Pagamentos automáticos
- Acompanhamento de projetos
- Portal completo do cliente

O sistema possui áreas segregadas para **Administradores** e **Clientes**, integradas ao mesmo banco de dados.

- Perfis MEI/Empresa

### 📝 Blog
- Categorias e comentários

### 📧 Newsletter
- Campanhas em massa

- Estatísticas
- Importação/exportação CSV

### 📄 Contratos
- Adesão / Renovação / Cancelamento
- PDF automático
- Preview em tempo real
- Multas automáticas

### 👥 Clientes
- PF/PJ
- Usuário automático
- Histórico de atividades

### 🤝 Indicações
- Código automático
- Desconto 10%
- Ranking de indicadores

### 💰 Pagamentos
- Mercado Pago
- PagBank
- PIX e Cartão
- Webhooks
- Faturas automáticas

### 📊 Status de Sistemas
- 21 status
- Percentual de progresso
- Histórico completo
